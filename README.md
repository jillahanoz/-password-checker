# Password Strength Checker 🔐

A Python-based web application that evaluates password strength in real-time and provides actionable feedback to help users create highly secure passwords.

## 🚀 Features

* **Real-time Evaluation:** Checks password strength instantly as the user types.
* **Visual Strength Indicator:** Dynamic feedback indicating whether a password is Weak, Medium, or Strong.
* **Feedback System:** Highlights exactly what security elements are missing.
* **Backend Validation:** Powered by a robust Python backend to handle validation logic cleanly.

## 🛠️ Built With

* **Python:** Core programming language.
* **Flask:** Lightweight web framework used to serve the application and handle routes.
* **HTML5 / CSS3:** User interface design and layout.

## 📋 Criteria Monitored

The application checks for the following security requirements:
1. Minimum length of 8 characters.
2. Contains at least one uppercase letter (`A-Z`).
3. Contains at least one lowercase letter (`a-z`).
4. Contains at least one number (`0-9`).
5. Contains at least one special character (e.g., `@`, `#`, `$`, `!`).
